<decisions_made>

- NoTransition for nested ScreenManager: SlideTransition caused touch handling bugs on Pi, menu items became unresponsive after navigating back from sub-screens
- KivyMD 1.2 workarounds: MDSwitch active deferred, no colorpicker, unicode symbols replaced with text
- async callbacks: LiveStatusCard supports both sync and async callbacks; hardware calls must use `async def` + `await`, never `asyncio.run()` inside Kivy callbacks
- HoldButton color fallback: `self.md_bg_color or NEAR_BLACK` because KivyMD initializes md_bg_color lazily (None at construction)
- Auth flow: PIN required on entry to debug screen, stays authenticated during internal navigation, resets on leaving to customer screen
- Brightness stored as 0.0-1.0 fraction in config, displayed as 0-100% via slider
- Machine ID format: "Vending-XXX" with numpad for number (001-999)
- All UI text in English (was originally German)
</decisions_made>

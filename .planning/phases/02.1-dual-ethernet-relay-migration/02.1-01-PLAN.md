```python
def modbus_crc(data: bytes) -> int:
    crc = 0xFFFF
    for byte in data:
        crc ^= byte
        for _ in range(8):
            if crc & 0x0001:
                crc = (crc >> 1) ^ 0xA001
            else:
                crc >>= 1
    return crc
```

2. Create `monitoni/hardware/modbus_tcp_relay.py`:
   - Class `EthernetRelayController(RelayController)` with persistent TCP connection.
   - Constructor: `__init__(self, host: str, port: int = 502, slave_address: int = 1, timeout: float = 1.0, max_channels: int = 8)`
   - `async connect()`: Uses `asyncio.wait_for(asyncio.open_connection(host, port), timeout=timeout)`. Sets status to CONNECTED on success, ERROR on failure.
   - `async disconnect()`: Closes writer, sets status DISCONNECTED. Stops reconnect loop.
   - `async _send_frame(self, frame: bytes) -> bytes`: Sends frame via writer, reads response via `asyncio.wait_for(self._reader.readexactly(8), timeout=self.timeout)`. Wrapped in `self._lock` (asyncio.Lock). On error: set status ERROR, clear reader/writer, return b''. CRITICAL: Do NOT retry on failure — return failure to caller (safety: avoid double-firing coils).
   - `async set_relay(channel, state)`: Uses `build_write_coil_frame()` from modbus_utils, sends via `_send_frame()`. Channel range: 1 to max_channels. Updates `_relay_states` cache on success.
   - `async get_relay(channel)`: Returns from `_relay_states` cache (same pattern as existing ModbusRelayController).
   - `async set_all_relays(state)`: Uses `build_write_all_coils_frame()`, sends via `_send_frame()`. Updates all cache entries.
   - `async health_check()`: Returns True if status is CONNECTED and writer is not None.
   - `async start_reconnect_loop()`: Background asyncio task. Only runs when status is ERROR or DISCONNECTED. Backoff: [2, 5, 10, 30] seconds. Only reconnects transport — never replays commands. Store task reference for cancellation.
   - `async stop_reconnect_loop()`: Cancel the reconnect task.
   - Property `connection_info -> dict`: Returns {host, port, status, last_error} for UI display.

   CRITICAL per research: Waveshare transparent mode = raw Modbus RTU frames with CRC over TCP. Do NOT use MBAP headers. Do NOT use pymodbus.

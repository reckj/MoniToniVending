   - If `use_mock`:
     - Create `MockDigitalInputController()` if method is modbus_di, else `MockSensorController()`.

   - `lock_door(level)`: Use `self.relay_levels` instead of `self.relay`. Guard: `if not self.relay_levels: return False`.

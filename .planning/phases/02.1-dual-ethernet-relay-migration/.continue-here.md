## Key Technical Notes
- Waveshare "transparent mode" = raw Modbus RTU frames (with CRC) over TCP socket — NOT Modbus TCP MBAP
- 30-CH module: coils 0x0000-0x001D, all-relay 0x00FF
- 8-CH Module C: coils 0x0000-0x0007, DI via FC02 0x0000-0x0007
- Both default to 192.168.1.254:502 — need different IPs in deployment
- Use readexactly() not read() for TCP responses to avoid partial reads
- Per-module asyncio.Lock, not global lock
